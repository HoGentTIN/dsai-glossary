## Probability theory

See also [probability](probability.md)

|     Symbol     | Interpretation                                           |
| :------------: | :------------------------------------------------------- |
|    $\Omega$    | universe                                                 |
| $A, B, \ldots$ | events (with $A \in \Omega$, $B \in \Omega$, etc.)       |
|     $P(A)$     | the probability of event $A$ (with $0 \leq P(A) \leq 1$) |
|    $P(A|B)$    | the probability of $A$ if $B$ occurs                     |
