## Waarschijnlijkheidsleer

Zie ook [kans](kans.md).

|    Symbool     | Betekenis                                                         |
| :------------: | :---------------------------------------------------------------- |
|    $\Omega$    | universum                                                         |
| $A, B, \ldots$ | gebeurtenissen (met $A \subset \Omega$, $B \subset \Omega$, enz.) |
|     $P(A)$     | de kans op gebeurtenis $A$ (met $0 \leq P(A) \leq 1$)             |
| $P(A&#124;B)$  | de kans op $A$ als $B$ zich voordoet                              |
